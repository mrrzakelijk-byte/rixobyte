# Rixobyte website

Statische website voor Rixobyte.

## Inhoud

- Homepage met diensten, werkwijze, projecten en offerteformulier
- Info-pagina
- NL/EN taalwissel met vlagbestanden
- Rixobyte-logo en favicon
- Projectcase: Rijschool Arend
- Contactformulier via Web3Forms

## Publiceren

Deze website kan direct gehost worden via Cloudflare Pages, GitHub Pages of Netlify.

Projecten bevatten nu Rijschool Arend en Akaareno.
